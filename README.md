# some-project-vue3

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build:prod
```

### Compiles and minifies for local serve
```
npm run build:local
```

### Previewing Locally
```
npm install -g serve
serve -s dist
```
